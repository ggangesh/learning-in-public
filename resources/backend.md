# Backend Resources

## GraphQL

### ⭐⭐⭐ Best Resources
- [How to GraphQL](https://www.howtographql.com/) - Complete tutorial
  - **What's great:** Step-by-step with multiple languages
  - **What's missing:** Doesn't cover subscriptions well

### ⭐⭐ Good Resources
- [Apollo Docs](https://www.apollographql.com/docs/) - Official documentation
  - **What's great:** Comprehensive API reference
  - **What's missing:** Too much at once, overwhelming
