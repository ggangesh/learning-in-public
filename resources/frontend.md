# Frontend Resources

> **Curated resources that actually helped me learn frontend development.**

---

## React

### ⭐⭐⭐ Best Resources
- [React Official Docs](https://react.dev/) - New documentation
  - **What's great:** Interactive examples, well explained
  - **What's missing:** Advanced patterns could be deeper

### ⭐⭐ Good Resources
- [Epic React](https://epicreact.dev/) - Kent C. Dodds course
  - **What's great:** Practical exercises
  - **What's missing:** Expensive

---

## Next.js

### ⭐⭐⭐ Best Resources
- [Next.js Docs](https://nextjs.org/docs) - Official documentation
  - **What's great:** Great examples, up-to-date
  - **What's missing:** App Router vs Pages Router can be confusing

---

## TypeScript

### ⭐⭐⭐ Best Resources
- [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/) - Official docs
  - **What's great:** Comprehensive
  - **What's missing:** Real-world patterns

- [Total TypeScript](https://www.totaltypescript.com/) - Matt Pocock
  - **What's great:** Advanced patterns explained simply
  - **What's missing:** Free tier is limited
